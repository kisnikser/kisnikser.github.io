The problem of choosing a sufficient sample size is investigated. The problem of determining a sufficient sample size without taking into account the nature of the parameters of the model used is considered. It is proposed to use the probability function of the sample. Two approaches to determining a sufficient sample size through the likelihood function are considered. The proposed approaches are based on heuristics about the behavior of the likelihood function with a large number of objects in the sample. The correctness of the proposed approaches is proved under certain restrictions on the model used. A method for predicting the similarity function in the case of insufficient sample size is proposed. A computational experiment is being conducted to analyze the properties of the proposed methods.

Исследуется задача выбора достаточного размера выборки. Рассматривается проблема определения достаточного размера выборки без учета природы параметров используемой модели. Предлагается использовать функцию правдоподобия выборки. Рассматриваются два подхода к определению достаточного размера выборки через функцию правдоподобия. Предлагаемые подходы основываются на эвристиках о поведении функции правдоподобия при большом количестве объектов в выборке. Доказывается корректность предложенных подходов при определенных ограничениях на используемую модель. Предлагается метод прогнозирования функции правподобия в случае недостаточного размера выборки. Проводится вычислительный эксперимент для анализа свойств предложенных методов.
